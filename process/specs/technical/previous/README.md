# Previous System-Overview Versions

Move superseded system-overview documents into this folder when later versions
become the active reference.

Example:

- active: `SYSTEM_OVERVIEW_v4.md`
- archived here: `SYSTEM_OVERVIEW_v1.md`, `SYSTEM_OVERVIEW_v2.md`,
  `SYSTEM_OVERVIEW_v3.md`

The active system-overview version should stay in the main
`workflow/specs/technical/` folder. Older versions can move here for historical
reference.
