# Product Decision Log: <product name>

> Artifact type: `durable product context artifact`

## Purpose

This log records durable product-level decisions and the reasoning behind them.

Use it to capture decisions that affect product intent, scope, capability
direction, user fit, or product constraints.

Do not use it for:

- milestone tracking
- implementation steps
- low-level technical-design reference docs
- system-overview version planning
