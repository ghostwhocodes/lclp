# Bootstrap Runtime Layer

Use this directory for bootstrap-only runtime artifacts when installing or
retrofitting the process into an existing repository.

The retrofit prompt pack itself lives under `bootstrap/`.
This directory is only for temporary runtime artifacts created while following
those prompts.

Typical contents may include:

- copied source proposals being mapped into the new workflow
- temporary bootstrap notes
- bootstrap inventories of existing docs or resources

Do not treat this as a normal steady-state planning layer.
